# Применение нейросетей для статистической обработки информации

## Краткое описание

Репозиторий содержит воспроизводимую цепочку подготовки, проверки качества и первичного анализа неоднородного набора данных. Прикладной набор связан с наблюдениями за изменением береговой бровки Волгоградского водохранилища, однако основной предмет работы здесь не сама экологическая интерпретация, а методы статистической и нейросетевой обработки информации.

На текущем этапе проект показывает, как из разнородных исходных таблиц формируется аккуратный производный слой, пригодный для описательного анализа, интервальных расчётов и сравнения профилей внутри участков. В базовое сравнение добавлен первый нейросетевой ориентир `MLPRegressor`, а отдельный демонстрационный автоэнкодер используется для поиска подозрительных наблюдений.

## Что уже сделано

- разобраны и нормализованы исходные таблицы по участкам, профилям, береговым наблюдениям, ветру, воде и базисным точкам;
- сформирован воспроизводимый производный слой с полями происхождения данных и контекстом проверки качества;
- рассчитаны интервальные метрики изменения береговой бровки;
- подготовлен безопасный поднабор для первого аналитического этапа;
- построены графики по периодам наблюдений, интенсивности изменений и межпрофильным корреляциям;
- добавлено базовое моделирование на производном датасете, включая первый нейросетевой ориентир `MLPRegressor`;
- добавлен демонстрационный нейросетевой поиск подозрительных наблюдений через автоэнкодер;
- добавлены QC-флаги и текстовые ограничения интерпретации.
